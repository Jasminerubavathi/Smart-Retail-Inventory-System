# Smart-Retail-Inventory-System
Web-based Smart Retail Inventory Management System using JSP, Servlet, MySQL


PROBLEM STATEMENT

Many small retail stores manage inventory manually, which makes real-time stock tracking difficult. 
This leads to stock shortages, product wastage due to poor rotation
As a result, stores face revenue loss and reduced customer satisfaction.
They lack,

❌ No real-time stock tracking
❌ Sudden stock shortages
❌ No proper FIFO rotation
❌ No automated alert system
❌ Product wastage and revenue loss
 

 
💡 Introduction

The Smart Retail Inventory Management System is a web-based application designed to help small and mid-sized retail stores manage their inventory efficiently and intelligently.

The system provides:

Real-time stock monitoring

Automated low-stock alerts

FIFO-based stock rotation for perishable goods

Efficient product and inventory management

By automating stock control and monitoring, the system reduces wastage, prevents stock shortages, and improves overall operational efficiency.

🚀 Core Features
✅ 1. Stock Management

Add new products

Edit product details

Delete products

Maintain quantity and price

Set threshold level for each product


✅ 2. Threshold Alert System

If product quantity becomes less than the defined threshold:

The system automatically generates an alert notification for Admin and Staff.

This helps prevent sudden stock shortages.


✅ 3. FIFO-Based Stock Rotation

For perishable products:

The system follows First In First Out (FIFO) logic

Older stock is sold first

Reduces product expiry and wastage

✅ 4. Role-Based Access

Admin: Full access (Manage stock, staff, alerts)

Staff: Limited operational access
